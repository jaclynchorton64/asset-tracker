# Asset Tracker

A Python web application for tracking IT assets, built with Streamlit.

## Features

- Add new assets with full details
- View all assets in a table
- Search assets by serial number
- Update asset status
- Delete assets
- Data stored in CSV format

## Fields Tracked

- Asset Name
- Serial Number
- Model
- Manufacturer
- Location
- Status (Active, In Repair, End of Life, Retired)
- Date Added

## How to Run

1. Clone the repository
2. Install dependencies with `uv add streamlit`
3. Run the app with `uv run streamlit run asset_tracker.py`

built in Microsoft Access (2001) and Microsoft Power Apps — now rebuilt in Python.
